3. for getting student details

/api/studinfo/<int:student_id> (get)

5. for getting issues user reported

/api/report/<int:user_id> (get)

6. for getting all the reports (for instructor and ta)

/api/view_reports (get)

7. for getting events

/api/events (get)

8. for getting tasks by user

/api/tasks/<int:user_id> (get)

9. for adding tasks by user

10. for deleting tasks

/api/tasks/delete/<int:task_id> (delete)

11. for getting courselist and marks to display on dashboard 

/api/mycourses/<int:student_id> (get)
